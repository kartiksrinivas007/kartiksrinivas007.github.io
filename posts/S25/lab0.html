Lab 0 - Takeaways

  • int to unsigned int and vice versa is just re-interpretation of the underlying bits
  • Beware implicit casting during function calls
  • The result of && <= >= etc are all assumed to be of type int (bool)
  • To compute ceil(u, 2^k) do (u + (1 <<k) - 1 >>k), else it rounds down and gives you the floor
  • Signed extension replicates bits
  • Truncation drops bits
  • Same size changes just reinterpret the bits
